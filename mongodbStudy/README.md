# 工程简介
以springboot为基础创建MongoDB的增删改查demo
# 延伸阅读
接口文档：

根据名字获取用户信息： 

  URL：http://localhost:8080/user/get_user
  请求类型：GET
  请求参数：{"name":"zhangsan"}

根据名字更新用户信息：

    URL：http://localhost:8080/user/update_user
    请求类型：GET
    请求参数：{"name":"zhangsan"}


根据名字删除用户信息：

    URL：http://localhost:8080/user/del_user
    请求类型：GET
    请求参数：{"name":"zhangsan"}
